# 💻 Blog Tecnológico

Um blog moderno e responsivo desenvolvido como projeto acadêmico para explorar as bases do desenvolvimento web. O site centraliza notícias sobre o mundo da tecnologia, com seções específicas para o mercado de jogos e dispositivos móveis.

## 🔗 Demonstração
O projeto está publicado e pode ser acessado aqui: 
👉 [Visualizar Blog Tecnológico](https://devarthurlima.github.io/blog-tecnologica/index.html)

## 📑 Estrutura do Projeto
O site é dividido em três áreas principais de conteúdo:
- **Página Inicial (Home):** Destaques e notícias gerais de tecnologia.
- **Seção de Jogos:** Conteúdo focado em lançamentos e novidades do mundo gamer.
- **Seção de Smartphones:** Artigos sobre os últimos lançamentos de celulares e hardware.

## 🚀 Tecnologias Utilizadas
Este projeto foi construído utilizando as tecnologias fundamentais da web:
- **HTML5:** Estruturação semântica de todas as páginas.
- **CSS3:** Estilização, layout e design responsivo.
